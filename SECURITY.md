# Security Policy

## Supported Versions

As of May 2022 (and until this document is updated), only the v1.x.x _stable_ releases of Isomera are supported for updates and bug fixes. Any previous versions are currently not supported and users are advised to use them "at their own risk".

- v1.x.x is considered LTS until further notice

## Reporting a Vulnerability

Please report (suspected) security vulnerabilities to
**[security@isomera.org](mailto:security@isomera.org)**.

You will receive a response from us within 72 hours. If the issue is confirmed,
we will release a patch as soon as possible depending on complexity
but historically within a few days.
